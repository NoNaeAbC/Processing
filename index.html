Was ist Processing?

Processing ist ein Open-Source Projekt von der Processing Foundation, zum Programmierenlernen. Es ist eine auf Java basierende Programmiersprache – die Syntax sollte also nicht ein Problem darstellen – inklusive IDE(nich greade die schönste). Processing besitzt eine vereinfachte Syntax, z.B. im Bereich der Klassen, und eine eigende 2D GUI Bibliothek, sowie ein par andere. Der Vorteil man beginnt sofort mit dem Fenster und den Grafiken.
